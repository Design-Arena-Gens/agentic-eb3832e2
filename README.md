# Деревянная бочка

Минималистичная статическая страница с иллюстрацией деревянной бочки, выполненной только при помощи HTML и CSS.

## Быстрый старт

1. Откройте `index.html` в современном браузере.
2. Наслаждайтесь объёмной бочкой без использования изображений и сторонних библиотек.

## Развёртывание на Vercel

Проект — статический. Для публикации достаточно выполнить:

```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-eb3832e2
```

После успешной загрузки проверьте работу по адресу `https://agentic-eb3832e2.vercel.app`.
